# skill-swd-chart
Create interactive chart visualisations in Claude Chat following Storytelling With Data best practices.

## Install
1. Click **Code > Download ZIP** from the Github repo
2. In Claud Desktop, go to **Settings > Customize > Skills**
3. Upload the ZIP file as a skill

## Usage
1. From any Claude Chat session, upload your data (CSV, Excel files) or ask Claude to return data from web searches or MCP connectors
2. Run `/swd-chart` and give it instructions of the story you are trying to visualise. For example, `/swd-chart visualise team performance from the CSV attachment, tell the story of what's going well, what needs improvement and actionable next steps.`
